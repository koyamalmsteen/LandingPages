Mineyama Geomagnetic Field Data

This data set is a part of the collection of geomagnetic field data at World Data Center for Geomagnetism, Kyoto (WDC-Kyoto). Mineyama is located ~100 km north of Kyoto.

General Characteristics
Collections: World Data Center for Geomagnetism, Kyoto
Project: World Data System
Instrument: Shimadzu MB162/MB163 Triaxial Fluxgate Magnetometer
Parameters: Geomagnetic Field in HDZ coordinates
Processing level: Provisional
Format: Kakioka binary format, WDC Format, IAGA-2002 Format
Coverage
Latitude: 35.57 deg
Longitude: 135.05 deg
Temporal resolution: 1 second
Start date: 1994-01
Stop date: Present
